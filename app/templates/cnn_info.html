CNN Visual Pipeline

Trace the mathematical operations mapping user handwriting matrices to output classes probability vectors.

IN
1. Input Image Receives a 28×28 grayscale bounding-box centered handwriting matrix containing pixel values normalized between 0.0 (black background) and 1.0 (white strokes).
CONV
2. Convolution Layer (Conv2D) Slides 3x3 weight filter matrices across the input pixels, computing mathematical dot-products to extract structural feature maps (curves, straight lines, corners).
RELU
3. ReLU Activation Applies Rectified Linear Unit f(x) = max(0, x) to introduce non-linear parameters, thresholding all negative activation values to zero to model complex boundaries.
POOL
4. Max Pooling Layer Downsamples feature resolution by selecting the maximum value in consecutive 2×2 blocks, reducing spatial dimensions, parameters weight, and improving translation invariance.
FLAT
5. Flatten Layer Flattens the high-level 2D matrices into a continuous 1D numerical array of nodes, enabling connection to standard feedforward classification weights layers.
FC
6. Fully Connected Dense Layer Interconnects all flattened inputs to 128 hidden nodes, combining detected shapes, loops, and contours features to vote on the class coefficients.
OUT
7. Output Classification Layer (Softmax) Maps dense inputs to 10 output class logits using the Softmax equation, generating a probability distribution representing the classification confidence for classes 0 to 9.
